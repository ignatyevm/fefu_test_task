## Стек
* Python 3.10
* FastAPI
* SQLAlchemy

## Запуск
```bash
git clone https://github.com/ignatyevm/fefu_test_task fefu_test_task
cd fefu_test_task
docker-compose up
```
Эндпоинт сервиса ```http://localhost:8000/```
